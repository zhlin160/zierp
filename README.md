# TP5开发的AUTH权限系统
## 安装
- 下载本项目到本地
```
git clone https://github.com/ningbnii/auth_tp5.git
```
- 导入数据库，数据库文件为根目录下面的tp10.sql
- 修改数据库配置文件

- 本地配置虚拟机，将虚拟机根目录指向public目录下面，访问配置的域名或者localhost，如果显示登陆页面则安装成功
- 登陆用户名：admin 密码：123456
